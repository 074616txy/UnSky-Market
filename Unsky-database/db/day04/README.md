# Day04 - 学生身份认证

## 当前文件说明

| 文件 | 说明 |
|------|------|
| `003_student_cert_schema.sql` | 创建学生认证表 `student_cert` |

## 表作用

`student_cert` 表用于保存学生身份认证申请信息，记录用户提交的学生姓名、学校、学号、证件图片路径、审核状态与审核备注。

## 核心字段说明

| 字段 | 说明 |
|------|------|
| `user_id` | 关联用户ID |
| `student_name` | 学生姓名 |
| `school` | 学校名称 |
| `student_id` | 学号 |
| `id_card_front` | 证件正面图片路径 |
| `id_card_back` | 证件反面图片路径 |
| `status` | 认证状态：0=待审核，1=审核通过，2=审核拒绝 |
| `remark` | 审核备注 |
| `create_time` | 申请时间 |

## 阶段说明

本阶段只完成 `student_cert` 表的建表脚本，不包含测试数据初始化。

## 服务场景

该表将服务于以下功能：

- 提交认证申请
- 查询认证状态
- 管理员审核认证
